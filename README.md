# FPGA-Based-High-Speed-Trigger-System-for-High-Energy-Physics-HEP-
A Verilog-based DAQ pipeline for high-speed particle spectroscopy on the Zynq-7010. It features a digital moving average filter for noise reduction and an FSM-driven peak detector for real-time event triggering. Includes a 16k-bin hardware histogram using BRAM for low-latency pulse height analysis.
